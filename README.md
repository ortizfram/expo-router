# Expo Router Example

Use [`expo-router`](https://docs.expo.dev/router/introduction/) to build native navigation using files in the `app/` directory.

## 🚀 How to use

```sh
npx create-expo-app@latest tabs-app -e with-router
cd tabs-app
touch tsconfig.json
npx expo
remove babel plugin expo router
touch app/index.js
```
